# Calculadora de série por intervalo de episódios

Site estático para GitHub Pages.

## O que faz
- Pesquisa séries na TMDb
- Permite escolher a temporada
- Permite escolher episódio inicial e final
- Soma a minutagem do intervalo escolhido
- Calcula o valor com base em R$ por minuto

## Publicar no GitHub Pages
1. Envie os arquivos para um repositório no GitHub
2. Vá em Settings > Pages
3. Escolha a branch principal e a pasta raiz
4. Salve e abra o link do Pages

## Observação
A chave da TMDb está fixa no JavaScript. Em site estático, ela pode ser descoberta por quem inspecionar o código-fonte.
